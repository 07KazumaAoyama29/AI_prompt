# 好きなものを言語化して伝える文章を書く方法

## step1 自分の感想を言語化する(AIに言語化を手伝ってもらう)

------プロンプト0--------
```txt
以下のことについてどう思う？

今回見た作品は○○です。
[url]

①刺さったシーン／歌詞／表情／演出

②その瞬間の感想
```





## step2 新しいチャットで、文章を書き上げていく

------プロンプト1--------
```txt
以下の流れで文章を書いていこうと思うので、手伝ってください。

### Phase 0: 鑑賞直後・遮断
他人の意見に流されないように、他人のSNS・感想を見るのを我慢する

一時メモを取る:　誰に見せるわけでもなく、感じたことを箇条書きにする。

「他人の感想を見る前に」書いておくことが重要。

### Phase 1: 設計（書く前）
読者を決める:

誰に向けて発信したいのかあらかじめ決めておくと、話し方（専門用語の有無など）が変わってくる 。

①想定読者を決める（未来の自分でもOK）。

②伝えたいポイントを決める(最終的なゴール)


### Phase 2: 執筆（書き出し〜本文）
書き出しの戦略:

一番いいのは、とりあえず書いてしまって、後で修正するというもの。以下、書き出しのヒント

①よかった要素を描写する:
良かったシーン(曲ならどの歌詞か)を伝える。引用から入るのもアリ。
その次に、引用した要素について自分がどう感じたのか、感情の原因を併せて説明する。ここで重要なのは全体についてふわっとした感想を書くのではなくて、自分の気になった一つの要素について丁寧に書くのを優先すること。1つの要素について深掘りすることにより細分化され、オリジナリティが生まれる。

②自分語りをする
自分の体験を書く➡同じような場面を説明➡どのような点が同じだったのか、共通項について説明

③文脈で始める
自分の体験の共通点ではなく、自分の好みとの共通点や自分には想像できなっかった新しさがあるものの場合、文脈の説明から始めてみる。
文脈というと難しそうだが、そうでもない。以下、例。
- 今自分が推しているアイドルは、実は松田聖子さんとの共通項がある
- 今自分が推しているプリンは、これまでのコンビニスイーツにはない新しさがある。

④問いで始める
ここまで紹介してきた書き出し法を試しても難しい場合は、問いから始めてみる。
なぜ？やどこが？といった疑問を提示する形で始める。

⑤展開（細分化と妄想）:

細分化: 「ヤバい」で終わらせず、対象を細かいトピックに分解して記述する。

妄想（解釈）: 「なぜここが良いと思ったのか？」に対し、正解を気にせず思考の大風呂敷を広げて解釈を書く。

主語を自分にする: 特に批判的な内容や強い感情を含む場合は、一般論（みんなこう思うはず）ではなく「私はこう思った」と自分を主語にする。

### Phase 3: クロージング（書き終える）
読んでいて冗長になる説明は避ける。
特に、伝えたいポイントに関係ないものは書くべきではない。

調べてわかることは削る:　自分の熱量や解釈が乗っていない情報はノイズになるため。

とにかく完走する:　書くネタが切れたら、以下のことをやってみる。
①書き出しがおかしくないかチェック
そもそも書き出しがおかしい場合は、特に広がりのない論点について話していることが多い。
②もういちど、書く対象に触れてみる。
曲に対する感想を書いているのならば、もう一度曲に触れてみることで新しい発見が見つかったり、文章の糸口が見えてきたりする。
③自分の好きな文章に触れてみる
なんとなく、微妙な文章しか書けないと感じたら、自分が最高だと思う文章に触れてみる。そうすとでこういう風に文章を書けばいいのかとなるかもしれない。

### Phase 4: デバッグ（修正）
修正をすればするほどプロの文章に近づく。
また、修正を前提にして文章を書くクセをつけると、文章を書き終えるハードルが下がる。

-別人になって、ポイントが伝わっているか確認
想定した読者に、伝えたいところが伝わる文章かどうかという軸で、文章を見直し・修正する。
書き終わってから、少し時間を空けるとよい。
- 文章の順番を変える
SNSのような短い文章から、長文まで幅広く使える方法。
よく、文章の構成をよくしろみたいな話があるが、それはほぼイコールで文章の順番ををよくしろということ。
文章が微妙だと感じたときは、一番いいフレーズを先頭に持ってくるとよい。
- いらない文章を削る
最終的な目標(伝えたいポイント)に関係のない冗長な文章を削除する。
- 見出しを付ける
ある程度長い、まとまった文章に使える方法。
長くて読みずらいと感じたときに使える
```
